CAE Web Full Stack Overhaul

A complete overhaul of the web apps used by the Computer Aided Engineering (CAE) Center at Western Michigan University. Originally written in Java and PHP, we rebuilt the site from the ground up using Python and the Django MVC framework, importing data where necessary. Functionality includes room management (scheduling, checkouts, and maintenance), work logging, employee shift management, and more.

Python, Django, HTML, CSS, Sass, JavaScript, React, MySQL

Pokébot

A simulation of the Pokémon games designed to run in the Discord environment. This bot reads messages from users and asynchronously performs certain actions based on the message content. Once a user catches a Pokémon, they can level it up, teach it new attacks, trade it with another user's Pokémon, use items on it, and more.

JavaScript, Node, MySQL

2018

Graph Colorer

A single page web application that attempts to color a randomly generated graph so that no two connected nodes have the same color. This project was created as part of a study on np-complete algorithms. Users can customize the graph generation by selecting the node count and the edge frequency. Once the graph is colored, the user will see how long it took and how many different colors were used.

JavaScript, HTML, CSS

2017

Hearthstone Clicker

A simple clicker game that runs in the browser. Themed around the popular card game Hearthstone, players earn points by clicking the card in the center. These points are used to purchase upgrades that increase overall point accumulation. The goal is to defeat the enemies portrayed on each card by earning a certain amount of points.

JavaScript, HTML, CSS

2016
